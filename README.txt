== Description ==

With this plugin you can create and customize a subscription form on your website where visitors can leave their details so that you can then export them freely and use them all the time for sending newsletter with external postmaster programs.This plugin have subscription form and it allows users to publicly subscribe by submitting their email address. You can add subscriptions form to your sidebar (HTML Widget) and post/pages with a [Shortcode]

**BASIC OPTIONS**  

* **Export XLS**  
You also have the option to export in XLS all the people who have been discharged and entan interested in receiving newsletter.

* **CUSTOMIZE YOUR FORM**  
The plugin has an area of customization of styles, so that always fit the same colors in your existing website.

* ** USE SHORTCODE **  
You can put shortcodes in both post , pages and widgets of the form that you created.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-great-newsletter` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your form on throught the plugin options page.
4. Copy and paste shortcode in your post, pages and HTML widgets


== Frequently Asked Questions ==

= Can I add shortcode in widget HTML?=

Yes, you can copy shortcode and paste in your post, page and HTML widget

= How can the user unsubscribe from the newsletter? =

If you want the user to have the ability to unsubscribe from the newsletter, it is necessary for enabling your unsubscribe link existing in the plugin options page.
